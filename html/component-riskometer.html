Riskometer CSS

CSS riskometer have multiple class level for risk from low to high-risk. You can add class on riskometer div low, low-moderate, moderate, moderate-high, high, very-high


You can also change size by adding style property --riskometer-width:200px 


            <div class="riskometer moderate mb-3">
                <!-- low low-moderate moderate moderate-high very-high high-->
            	<div class="indicator"></div>
            </div>
            
            <div class="riskometer moderate mb-3" style="--riskometer-width: 200px;">
                <!-- low low-moderate moderate moderate-high very-high high-->
                <div class="indicator"></div>
            </div>
